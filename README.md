# CurrencyBot 💵

Telegram-бот для получения актуального курса доллара к рублю.

## Команды
- `/start` — приветствие
- `/course` — показать курс доллара

## Технологии
- Python 3
- pyTelegramBotAPI
- BeautifulSoup4
- Requests

## Запуск
```bash
pip install pyTelegramBotAPI beautifulsoup4 requests
python bot.py
